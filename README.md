### 一个基于npm script工作流的简单页面

### 实现了如下的功能：
  1. 将 `es6` 转换成 `es5`,仅限于语法转换，不包含各种的 `polyfill`

  2. 可以使用 `sass` 写 `css`, 并且自动添加厂商前缀

  3. 保存代码浏览器自动刷新

  4. 对图片和svg进行压缩

  5. 整合了自己常用的手机端的适配方案


### 使用：
   1.  `git clone https://github.com/jackPanyj/npm-script-start.git`

   2.  `cd npm-script-start && npm i`

   3.  `npm start`
